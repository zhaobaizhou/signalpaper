Workflow Field Guide is a small reference library of AI workflows I actually use.

Not a prompt repository. Not a list of tips. A curated set of end-to-end workflows for specific situations that come up in knowledge work and publishing.

## What's inside

**Research workflows**
- Distilling long documents into decision-ready summaries
- Extracting key claims with citations
- Comparing multiple sources on a question

**Writing workflows**
- First-draft generation from structured notes
- Headline and opening paragraph iteration
- Final pass for clarity and specificity

**Publishing workflows**
- Platform-specific adaptation (newsletter, blog, LinkedIn)
- Metadata generation (title, description, tags)
- Cross-posting version control

## Format

Each workflow is a single markdown file with:
- Situation (when to use it)
- Inputs required
- Step-by-step procedure
- Example prompt
- Expected output

## Status

Shipped. Available as a free download. Used by ~200 people based on download count. No plans for major updates — the core workflows are stable.
