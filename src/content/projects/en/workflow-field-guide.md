Workflow Field Guide is a demo project for collecting repeatable working patterns.

It is a useful example for creators who want to publish practical methods without building a separate documentation site.

## What it contains

- Short playbooks
- Checklists
- Tool notes
- Publishing routines
- Maintenance steps
